1- If... else / si... sinon

Si... sinon

La struture conditionnelle "if... else" sera plus complète car on va exécuter un bloc de code si un test renvoi true ou un autre dans le cas contraire./p>

Titre

let x = 0.5;
if(x > 1) {
// si e test est vrai
alert('IF x contient ' + x + ' et cette valeur est supérieure à 1');
} else {
// sinon j'exécute celui-ci
alert('ELSE x contient ' + x + ' et cette valeur est inférieure à 1');
}
