JavaScript - III/09 - Boucles

Les boucles

Les boucles vont nous permettre d'exécuter plusieurs fois un bloc de code, c'est-à-dire d'exécuter un code en boucle tant qu'une condition donnée est vérifiée ; utiliser une boucle nous permet de n'écrire le code que l'on doit exécuter qu'une seule fois.

Il existe 6 boucles en JS

  1. La boucle while : tant que
  2. La boucle do... while : faire... tant que
  3. La boucle for... : pour... cf.
  4. La boucle for... in : pour... dans
  5. La boucle for... of : pour... parmi
  6. La boucle for... await... of : pour (en attente)... parmi

Le fonctionnement général des boucles est toujours le même : on pose une condition qui sera généralement liée à la valeur d'une variable et on exécute le code de la boucle "en boucle" tant que la ocndition est vérifiée.

Pour éviter de rester bloqué à l'infini dans une boucle, vous pouvez dons déjà noter qu'il faudra que la condition donnée soit fausse à un moment donné (pour pouvoir sortir de la bouche).

Pour que notre condition devienne fausse à un moment, on pourra par exemple incrémenter ou décrementer la valeur de notre variable à chaque nouveau passage dans la boucle (ou modifier la valeur de notre variable selon un certain schéma).

Les boucles vont donc être essentiellement composées de 3 choses :
  • Une valeur de départ qui sert à initialiser la boucle et qui servira de compteur.
  • Un test ou une condition de sortie qui précise le critère de sortie de la boucle.
  • Un "itérateur" (ex.: le w++ de l'exo) qui va modifier la valeur de départ de la boucle à chaque passage de la boucle jusqu'à ce que la condition de sortie soit vérifiée. Pour cela on devra "incrémenter" la valeur de départ. Souvent on utilise la lettre "i" pour "itérateur".

2- Opérateurs d'incrémentation & de décrémentation

Incrémenter
Ajouter 1 à une valeur
Décrémenter
Enlever 1 à une valeur
Exemple Résultat
++x Pré-incrémentation : incrémente la valeur contenue dans la variable x, puis retourne la valeur incrémentée
x++ Post-incrémentation : retourne la valeur contenue dans x avant incrémentation, puis incrémente la valeur de $x
--x Pré-décrémentation : décrémente la valeur contenue dans la variable x, puis retourne la valeur décrémentée
x-- Post-décrémentation : retourne la valeur contenue dans x avant décrémentation, puis décrémente la valeur de $x

Nos exemples

While

La boucle while, "tant que" en français, va nous permettre de répéter une série d'instructions tant qu'une condition donnée est vraie, c'est à dire tant que la onditon de sortie n est as veirifiee.

Do... while

La boucle "do... while", "faire... tant que" est relativement semblable à while, la différence réside dans l'ordre dans lequel vont se faire les opérateurs.
On effectuera un premier passage dans une boucle "do... white" même si la condition de sortie n'est pas vérifiée, donc le code sera executé au moins une fois.
