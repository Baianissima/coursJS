JavaScript - III/05 - Opérateurs logiques

Pour effectuer plusieurs comparaisons

Les opérateurs logiques

Ce type d'opérateur va nous permettre d'effecteur plusieures comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test

Présentation et utilisation des opérateurs logiques

Récap

Opérateurs logiques
Nom Symbole Description
AND (ET) && Renvoie true si toutes les comapraisons sont vraies ou false sinon.
OR (OU) || Renvoie true si l'une des comapraisons est vraie, ou false sinon.
NON (non) ! Renvoie true si une des comparaison est évalué à false sinon.
