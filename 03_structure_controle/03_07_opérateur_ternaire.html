Les structures conditionnelles ternaires

Les ternaires sont une autre façon d'écrire nos conditions en utilisant une syntaxe basée sur l'opérateur ternaire "?". Mais il y aller mollo aussi, comme pour l'usage du switch.

1 - Ecriture

On peut écrire la condition sur une seule ligne

Ex.: document.getElementById('p4').innerHTML = y > 10 ? 'y est supérieur à 10 ' : 'y est inférieur à 10';

Les ternaires se présentent sous la forme suivante : test ? code à executer si true : code à exécuter sinon (else, si c'est faux)
