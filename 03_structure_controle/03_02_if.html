1- If...

La condition if est l'une des plus utilisée et est également la plus simple à apprhénder puisqu'elle va nous permettre d'éxecuter un bloc de code si ET seulement si le résultat vaut true.

Attention ! : exceptions

Tout test d'une condition va être évalué dans un contexte booléen, donc à trueà l'exception des valeurs suivantes, qui vont être évaluées à false.

  • La valeur 0 (zéro)
  • Le boléen false
  • Une chaîne de caractères vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)
